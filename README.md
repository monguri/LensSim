# LensSim

Simulation of LensSystem

## Features

- [x] RayTrace
- [x] Focusing
- [x] Exit Pupil Sampling
- [x] Spectral Rendering with IBL
- [x] Lens Flare
- [x] Chromatic Aberration

### Lens Elements

* Aperture
* Spherical Lens

## Build

```
mkdir build
cd build
cmake ..
make
```
